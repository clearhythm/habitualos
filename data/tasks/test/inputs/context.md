# Test Card Generation Task

## Description
Generate 3 simple test cards for each user based on their profile data. This is a system test to verify the scheduled task execution engine works correctly.

## Instructions
For each user, generate 3 cards with the following structure:
1. **Card 1: Welcome Card** - A personalized greeting using their name and role
2. **Card 2: Daily Focus** - A motivational message based on their interests
3. **Card 3: Action Item** - A simple actionable task related to their goal

## Template Examples

### Example Card 1 (Welcome)
```
# Welcome, [Name]!

Hello [Name], you're doing great as a [Role].

Today is a new opportunity to make progress on your goals.

---
Generated on: [Current Date]
```

### Example Card 2 (Daily Focus)
```
# Your Focus Today

Based on your interest in [Interest], here's something to think about:

[1-2 sentences of motivation related to their interest]

Remember: Small steps lead to big changes.
```

### Example Card 3 (Action Item)
```
# Today's Action

Goal: [Their Goal]

**One thing you can do today:**
- [A specific, actionable task related to their goal]

You've got this!
```

## Additional Guidelines
- Keep cards concise and friendly
- Use markdown formatting
- Personalize based on the user data provided
- Keep the tone encouraging and supportive
- Each card should be self-contained
